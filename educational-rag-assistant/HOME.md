# Educational RAG Assistant

Интеллектуальный помощник для студентов ИТ-специальностей на основе технологии Retrieval-Augmented Generation (RAG).

---

## Описание проекта

Educational RAG Assistant — система, которая помогает студентам изучать учебные материалы через анализ загруженных документов. Вместо ответов из интернета, ИИ работает исключительно с вашими файлами: методичками, лекциями и конспектами.

---

## Для кого этот проект

- Студенты технических вузов и колледжей
- Преподаватели для автоматизации ответов на типовые вопросы
- Исследователи в области искусственного интеллекта

---

## Технологии

| Технология | Описание |
|------------|----------|
| **RAG** (Retrieval-Augmented Generation) | Поиск информации в загруженных документах и генерация ответов на их основе |
| **FastAPI** | Современный фреймворк для создания высокопроизводительных API |
| **LangGraph** | Библиотека для построения графовых цепочек обработки запросов |
| **GLM-OCR** | Распознавание текста в изображениях и PDF-документах |
| **E5-large** | Модель для генерации текстовых эмбеддингов |
| **PostgreSQL** | Хранение метаданных сессий пользователей |
| **Qdrant** | Векторная база данных для хранения эмбеддингов |
| **Ollama Cloud** | Инференс LLM моделей |

---

## Основные возможности

- Загрузка документов: PDF, PNG, JPEG через Drag-and-Drop
- Семантический поиск: Найдите нужную информацию в тексте по смыслу
- Чат с ИИ: Задавайте вопросы и получайте ответы на основе ваших материалов
- Управление сессиями: Сохраняйте историю диалогов для каждой темы
- Настройка сложности: Адаптируйте тон ответов (Новичок / Средний / Эксперт)
